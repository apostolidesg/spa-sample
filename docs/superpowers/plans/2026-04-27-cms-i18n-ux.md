# CMS i18n UX Improvement Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Merge 12 Decap CMS collections into 6 using Decap's built-in i18n support, so the client edits both EN and EL in one form with a language tab switcher.

**Architecture:** Add a global `i18n` block to `config.yml`, change each collection's `folder` from `content/en` or `content/el` to `content`, add `i18n: true` to each collection, and add `i18n: duplicate` to the hidden `file_name` field. No changes to the Vue app or content JSON files.

**Tech Stack:** Decap CMS config (YAML)
